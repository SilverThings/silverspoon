Silverspoon.io example using Apache Camel to expose Temperature sensor as a REST endpoint.

This example shows how to use Servlet REST to define REST endpoints in Camel routes using the Rest DSL From a web browser you can access the REST service to retrieve temperature form the configured sensors:
  • temperature - to view the temperature

This example offers an API browser using Swagger which is accessible from the following link

  • api-docs - list all CamelContexts in the JVM that may have REST services
  • api-docs/temperature - temperature - overview of the apis from the REST services

To use the swagger ui, follow this link. Replace the URL value with this one to access the REST local resources http://localhost:8080/${artifactId}/api-docs

  • swagger - swagger ui page
